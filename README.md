# MakeFun 桌遊

---

## 前言

此為一款開源桌遊，目的是讓老師以及學生，實際動手做，體會Maker與STEAM的精神，而桌遊的所有資源，從自走循跡車、RFID圖卡、桌遊軌道以及場地皆提供檔案，歡迎自行下載，並動手DIY。

## 桌遊規則說明

根據關卡定義， 擺放對應的「感應軌道」在地圖圖板上， 並將機器人放置在關卡示意的起點區域， 過程中， 玩家透過「指令卡牌」在機器人身上進行感應， 讓機器人沿著軌道做出對應的動作， 進而到達目的地達成任務。
